Jan 11

Spent a good amount of time on the mongodb tutorial I found yesterday. Some tl;dr for the MEAN stack. MongoDB db is made of collections (~tables), which hold documents (~row). Biggest difference from relational db is lack of fixed schema and allowing for data structures such as arrays as values. Node.js is a Javascript server-side interpreter NOT a framework, whatever that means. Its defining desin is its async ops - no blocking! Express is the front-end web app framework. You create an app that has something called a middleware chain, and each link in the chain uses a req (request) and res (results) object to store results. Angular is another web framework for making HTML/JS-heavy web apps (didn't go through this since I wanted to learn about React).
